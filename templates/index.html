Tools & API

Access circRNA data via the following APIs:

  • /api/statistics: Get database statistics
  • /api/circRNAs: Get circRNA information
  • /api/diseases: Get disease associations
  • /api/functions: Get functional annotations
  • /api/query: Query data (parameters: table, key)
